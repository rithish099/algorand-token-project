# Algorand Token Creator

## 📌 Project Description
This project demonstrates how to create an Algorand Standard Asset (ASA) using Python SDK.

## 🚀 Features
- Connect to Algorand TestNet
- Create a new token (ASA)
- View transaction details

## 🛠️ Technologies Used
- Python
- Algorand SDK
- TestNet

## 📂 Project Structure
- main.py → Main program
- requirements.txt → Dependencies

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
